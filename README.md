## 🧭 What n8n-cli does

n8n-cli gives you a simple way to work with the n8n API from your computer. You can use it to manage workflows, check runs, update credentials, and work with nodes and webhooks.

It is built for people who want to control n8n from a command line tool without setting up Node.js. It uses Python and has no extra package chain to manage.

## ✨ Main features

- Works with n8n Cloud and self-hosted n8n
- Uses Python with no npm or Node.js needed
- Includes 80+ commands for daily n8n tasks
- Helps you manage workflows
- Lets you view and handle executions

## 🔐 Set up your n8n profile

n8n-cli can store more than one connection profile. This helps if you use both a test server and a live server.

Typical setup steps:

1. Create a profile name.
2. Add your n8n base URL.
3. Add your API key or token.
4. Save the profile.
5. Switch profiles when you need another n8n instance.

## 🧹 Troubleshooting

- If the app does not open, check that Windows allowed the file.
- If a command fails, confirm your n8n URL is correct.
- If login fails, check the API key or token.
- If you use a proxy or firewall, make sure it allows access to n8n.
- If a profile does not load, create it again with the right name and URL
